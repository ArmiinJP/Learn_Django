در هر اپی میای جدا برای خودش url هاشو تعریف میکنیم با هر اسمی که خواستی به عنوان ارگومان اول تابع path که مال کتابخانه from django.urls import path پاس میدی و تهشون هم / را میزاری.

همچنین view های هر اپی را هم جدا میای برای خودش تعریف میکنی.

در urls.py باید بیای view های این اپ را به url هایی که تعریف کردی با path وصلشون کنی و در لیست urlpatterns اضافه کنی.

---------------
برای اینکه url های اپهات در url اصلی پروژه قابل دسترسی باشه میری داخل url پروژه و مثلا دستور زیر را میزنی 
from django.urls import path, include
path('home/', include("home.urls"))
یعنی عملا include را آوردی و به کمکش url های این اپت را بهش پاس دادی.
الان دیگه home میشه سطح اول url و urlهایی که داخل این اپ تعریف میکنی میشه سطح دوم مثل hello مثلا: 
http://127.0.0.1:8000/home/hello/
اگه نخواستی چند سطحی بشه باید مسیر به path پاس ندی، یعنی در urls مربوط به سطح پروژه، باید مسیر زیر را بدی:
path('', include("home.urls"))
